# Module 1: Version Control

This module covers the fundamentals of Version Control and explains why it is an essential tool for software development.

## Topics

- What is Version Control?
- Why do we need Version Control?
- Types of Version Control Systems
- Centralized vs Distributed Version Control
- What is Git?
- What is GitHub?
- Git vs GitHub

## Learning Goal

By the end of this module, I should be able to explain the purpose of Version Control and understand the difference between Git and GitHub.
