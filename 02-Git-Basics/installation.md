# Git Installation

## Overview

Before using Git, it must be installed on the local machine. Once installed, Git can be used in any project without reinstalling it.

---

## Why Do We Install Git?

Git is a Distributed Version Control System (DVCS) that helps developers track project history, manage changes, and collaborate efficiently.

Installing Git provides the Git Command Line Interface (CLI), Git Bash, and integration with code editors like VS Code.

---

## Verify Installation

To check whether Git is installed, run:

```bash
git --version
```

Example Output:

```text
git version 2.51.0.windows.1
```

If Git is not installed, an error similar to the following will appear:

```text
'git' is not recognized as an internal or external command...
```

---

## Key Points

- Git is installed only once on the operating system.
- It can be used for multiple projects.
- The installed version can be verified using `git --version`.
