# Module 2: Git Basics

## 📌 Overview

This module covers the essential Git concepts required to start working with Git repositories. It focuses on configuring Git, creating repositories, understanding the Git workflow, and using the basic commands that every developer should know.

---

## 📚 Topics

- Git Installation
- Git Configuration
- Creating a Git Repository
- Understanding the `.git` Directory
- Git Status
- Git Workflow

---

## 📂 Files

| File | Description |
|------|-------------|
| `installation.md` | Installing Git and verifying the installation |
| `git-config.md` | Configuring Git for the first time |
| `git-init.md` | Initializing a Git repository |
| `git-status.md` | Understanding the current state of a repository |
| `git-workflow.md` | Working Directory, Staging Area, Local Repository, and Remote Repository |
| `first-repository.md` | Creating and managing the first Git repository |

---

## 🎯 Learning Outcomes

After completing this module, I will be able to:

- Install and configure Git
- Create and initialize Git repositories
- Understand how Git tracks file changes
- Use the basic Git workflow confidently
- Prepare projects for version control
