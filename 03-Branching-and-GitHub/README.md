# Module 3: Branching & GitHub

## 📌 Overview

This module covers Git branching, GitHub workflows, and collaboration techniques used in real-world software development. It explains how developers work on different features simultaneously, manage remote repositories, and collaborate efficiently using GitHub.

- GitHub Workflow
- Team Collaboration
